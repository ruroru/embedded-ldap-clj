# embedded-ldap-clj
embedded-ldap-clj is an embedded in-memory ldap server.

## Installation
Add embedded-ldap-clj to dependency list
```clojure
[org.clojars.jj/embedded-ldap-clj "1.0.0-SNAPSHOT"]
```

## Usage
```clojure
(:require
  [jj.embedded.ldap :as ldap])
```
